# 🌱 Smart Irrigation System

Arduino-based automatic irrigation using soil moisture sensor.

## 🛠️ Components
- Arduino UNO
- Soil Moisture Sensor (A0)
- Relay Module (Pin 7)
- 16x2 LCD with I2C
